## Setup

Ensure you have the following requirements:

- Python 3.8

Documentation has the following requirements:

- TeX Live
- Biber

First, create the virtual environment:

    python -m venv venv
    source venv/bin/activate
    pip install -U pip

    python -m project_pactum
